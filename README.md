# 2026 간사이 가족여행 가이드

마스터 문서 v3.3을 기준으로 제작한 GitHub Pages용 정적 여행 안내 사이트입니다.

## 공개 사이트

- https://huskyminu.github.io/26kansai_guide/

## 구성

- 2026년 7월 28일~31일 요일별 일정
- 오사카·교토 주요 여행지 사진
- 항공편·숙소·폭염 대응·삭제 우선순위
- 가족 의견 작성 후 휴대전화 공유 또는 복사 기능

## 배포

이 저장소의 `.github/workflows/pages.yml`이 `main` 브랜치 변경 시 GitHub Pages 배포를 자동 실행합니다.

저장소의 **Settings → Pages → Build and deployment → Source**가 `GitHub Actions`로 설정되어 있어야 합니다.

## 참고

정적 사이트이므로 가족 의견은 서버에 저장하지 않습니다. 의견 작성 후 휴대전화 공유 메뉴나 복사 기능을 이용해 가족 채팅방으로 전달합니다.

일정과 운영시간은 Master Plan v3.3 기준이며, 출발 전에 공식 자료로 다시 확인해야 합니다.
